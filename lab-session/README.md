# Computer Vision Lab Session

This directory stores hands-on materials derived from lecture slides.

## Organization

- One folder per lecture topic
- `data/` keeps small assets used by the lecture
- `examples/` keeps runnable Python scripts
- `*.ipynb` keeps guided notebook versions

## Current labs

- `01-opencv-practice/`: image I/O, grayscale conversion, resize, drawing, webcam, and mouse interaction with OpenCV
- `02-image-processing-basics/`: thresholding, morphology, gamma correction, histogram equalization, filtering, and geometric transforms
- `03-edge-and-region/`: Sobel and Canny edge detection, contours, Hough transform, superpixels, and region features
- `04-local-features/`: Harris corner response, SIFT keypoints, and FLANN-based feature matching
- `05-vision-agent/`: PyQt-based vision apps for control panels, webcam capture, GrabCut interaction, alert logic, and visual effects
- `06-deep-learning-vision/`: MNIST classification, LeNet, augmentation, pre-trained ResNet inference, Mask R-CNN, and YOLO

## Legacy files

The root-level OpenCV scripts are kept as the original quick experiments.
New lecture materials should be added inside lecture-specific folders.
