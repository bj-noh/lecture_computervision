# lecture_computervision

Computer Vision course materials organized around a `lab-session` main directory.

## Structure

- `lab-session/01-opencv-practice`: OpenCV basics, image I/O, drawing, webcam, and mouse events
- `lab-session/02-image-processing-basics`: thresholding, morphology, filtering, histogram equalization, and geometry transforms
- `lab-session/03-edge-and-region`: edges, contours, Hough transform, superpixels, and region features
- `lab-session/04-local-features`: Harris corners, SIFT keypoints, and FLANN matching
- `lab-session/05-vision-agent`: PyQt-based vision application examples

## Recommended packages

```bash
pip install opencv-python matplotlib numpy notebook scikit-image PyQt5
```

Each lecture folder contains its own `README.md`, notebook, and runnable example scripts.
